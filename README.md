
# 🌦️ Linux Weather Automation Project

A Linux-based automation and web application project that retrieves and displays weather data using Bash, Python, and a Flask web interface.

This project started as a simple Bash automation and was later extended into a small web application with API integration and service automation.

---

## 🚀 Project Overview

- Automated weather data collection
- REST API integration (yr.no)
- Flask web application
- Bash automation
- Systemd service integration
- Logging and reporting

---

## 🛠️ Technologies Used

- Linux (AlmaLinux)
- Bash
- Python 3
- Flask
- REST API (yr.no)
- systemd
- tmux
- curl
- Git

---

## 📂 Project Structure

- `app.py` – Flask application
- `weather.sh` – automation script
- `templates/` – HTML templates
- `reports/` – generated data
- `logs/` – runtime logs
- `docs/` – documentation & screenshots

---

## 🌐 Features

- Web interface for weather display
- Live data from yr.no API
- JSON data processing
- HTML rendering with Flask
- Background service execution
- Automated execution using systemd
